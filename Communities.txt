1. Client and Problem

Client

Purpose Mindset Initiative — Akhtar Badshah + QuestionPro.

Platforms Used

  · QuestionPro Surveys

  · QuestionPro Communities

  · AI Router using the OpenAI integration

Background

The Purpose Mindset exercise is designed to help individuals articulate their personal purpose by reflecting on two core areas: core strengths and foundational values.

The experience needed to feel guided, reflective, premium, and personalized. The end goal was not just to collect survey responses, but to help each participant walk away with a generated personal purpose statement and a clear pathway for further engagement through a community and workshops.

Specific Use Case

The team wanted to create a simple but powerful survey-led flow where users could:

  1. Select their top 3 strengths from a predefined list.

  2. Select their top 3 values from a predefined list.

  3. Receive a generated personal purpose statement.

  4. Be classified as either Proactive or Responsive.

  5. Be invited into a QuestionPro Community and future workshops.

Key Requirements

  · Work inside the QuestionPro survey flow.

  · Generate the output in real time.

  · Allow the prompt to be edited without engineering dependency.

  · Deliver a premium and guided experience for the participant.

  · Integrate with QuestionPro Community for panel recruitment and continued engagement.

2. Solution Overview

The solution combines QuestionPro Surveys, Communities, custom JavaScript, and AI Router.

At a high level, the flow works as follows:

  1. A QuestionPro Community is created for the Purpose Mindset initiative.

  2. The survey collects participant profile information, strengths, and values.

  3. Selected strengths and values are stored in custom variables.

  4. A hidden survey question triggers a webhook to AI Router.

  5. AI Router generates a purpose statement and classifies the participant as Proactive or Responsive.

  6. The generated output is saved back into custom variables.

  7. A final presentation/report screen displays the personalized purpose statement, purpose driver, golden nugget, strengths, and values.

  8. The participant can download the report as an image.

  9. The participant can be moved into the Purpose Mindset Community and invited to workshops.

3. Implementation Steps

Step 1: Create the Community

Create a dedicated QuestionPro Community for the Purpose Mindset initiative.

Community URL: https://purpose.questionpro.com

Community Setup Requirements

  · Follow the Purpose Mindset brand guidelines.

  · Add all relevant survey questions as Community profile fields.

  · Ensure that answers captured in the survey can be mapped and linked to members inside the Community.

  · Use the Community as the engagement layer after the participant completes the purpose exercise.

Why This Matters

The Community is not just a destination after the survey. It becomes the engagement hub where participants can continue their purpose journey, join workshops, and be segmented based on their generated purpose profile.

Create a consumer and use case inside AI Router.

The AI Router use case is responsible for receiving the participant’s selected strengths and values, generating the personal purpose statement, categorizing the participant as Proactive or Responsive, and returning the generated output back to the survey.

6. AI Router Webhook Trigger

After the strengths and values are selected and stored in custom variables, add a hidden question in the survey. This hidden question is used to trigger the webhook to AI Router.

Purpose of the Hidden Question

The hidden question acts as the processing step between the survey inputs and the generated output. It sends the participant’s selected strengths and values to AI Router and receives the generated purpose output back.

Webhook Output Storage

The returned values from AI Router are saved into custom variables. In this implementation:

  · Generated purpose statement is saved in custom30.

  · Persona type is saved in custom31.

8. Potential Iterations

The same flow can be expanded into an iterative loop. For example, if the user is not satisfied with the generated purpose statement, the survey can ask for additional details, send those details back to AI Router, and generate a refined purpose statement.

Iterative Flow Example

  1. User receives generated purpose statement.

  2. User is asked: “Are you happy with this statement?”

  3. If no, user provides more details or feedback.

  4. The additional feedback is sent back to AI Router.

  5. AI Router generates a more accurate or personalized purpose statement.

  6. Updated output is displayed in the final report.

How to Set This Up

  · Capture the new user input in a custom variable.

  · Trigger AI Router again through another hidden question or webhook step.

  · Pass the original strengths, values, and refinement input into AI Router.

  · Save the refined purpose statement back into a custom variable.

  · Display the final refined statement in the report.

11. Download Report Logic

The report includes a Download Report button. When the participant clicks this button, the report section is converted into an image and downloaded as report.png.

JS Used in Pre Script

+--------------------------------------------------------------------------+
| $(document).ready(function() {                                           |
|                                                                          |
| $(".export-pdf").click(function() {                                      |
|                                                                          |
| $("#report").css("border-radius", "0px");                                |
|                                                                          |
| window.scrollTo(0, 0);                                                   |
|                                                                          |
| const { jsPDF } = window.jspdf;                                          |
|                                                                          |
| html2canvas(document.querySelector("#report"), {                         |
|                                                                          |
| scale: 5, // Higher scale for better quality                             |
|                                                                          |
| useCORS: true,                                                           |
|                                                                          |
| allowTaint: true,                                                        |
|                                                                          |
| logging: false,                                                          |
|                                                                          |
| letterRendering: true,                                                   |
|                                                                          |
| onrendered: function(canvas) {                                           |
|                                                                          |
| var ctx = canvas.getContext('2d');                                       |
|                                                                          |
| ctx.webkitImageSmoothingEnabled = false;                                 |
|                                                                          |
| ctx.mozImageSmoothingEnabled = false;                                    |
|                                                                          |
| ctx.imageSmoothingEnabled = false;                                       |
|                                                                          |
| },                                                                       |
|                                                                          |
| }).then(canvas => {                                                      |
|                                                                          |
| let imgData = canvas.toDataURL("image/png", 1.0); // Highest quality PNG |
|                                                                          |
| let link = document.createElement('a');                                  |
|                                                                          |
| link.href = imgData;                                                     |
|                                                                          |
| link.download = "report.png";                                            |
|                                                                          |
| document.body.appendChild(link);                                         |
|                                                                          |
| link.click();                                                            |
|                                                                          |
| document.body.removeChild(link);                                         |
|                                                                          |
| });                                                                      |
|                                                                          |
| $("#report").css("border-radius", "10px");                               |
|                                                                          |
| });                                                                      |
|                                                                          |
| });                                                                      |
|                                                                          |
| setTimeout(function() {                                                  |
|                                                                          |
| //document.forms['run'].onsubmit();                                      |
|                                                                          |
| }, 120000);                                                              |
+==========================================================================+
+--------------------------------------------------------------------------+

What This Script Does

  1. Waits for the page to load.

  2. Adds a click listener to the .export-pdf button.

  3. Temporarily removes the report border radius for cleaner export.

  4. Scrolls the participant to the top of the page.

  5. Uses html2canvas to capture the #report element.

  6. Uses a high scale value of 5 for better image quality.

  7. Converts the captured canvas into a PNG image.

  8. Creates a temporary download link.

  9. Downloads the file as report.png.

  10. Removes the temporary download link.

  11. Restores the report border radius.

Additional Note

The script also includes a commented auto-submit section:

+-----------------------------------------------------------------------+
| setTimeout(function() {                                               |
|                                                                       |
| //document.forms['run'].onsubmit();                                   |
|                                                                       |
| }, 120000);                                                           |
+=======================================================================+
+-----------------------------------------------------------------------+

This can be enabled if the survey needs to auto-submit after a fixed duration.

13. Summary

This implementation turns a simple survey selection exercise into a guided AI-powered purpose journey.

The participant selects their top strengths and values, AI Router generates a personalized purpose statement, and QuestionPro displays the output as a polished downloadable report.

The full solution uses:

  · QuestionPro Surveys for data collection.

  · Custom JavaScript for selection limits and custom variable mapping.

  · AI Router for purpose statement generation and participant classification.

  · QuestionPro Communities for continued engagement.

  · A presentation question for the final branded report.

  · html2canvas for report download.

This creates a premium, real-time, and repeatable Purpose Mindset experience while keeping prompt control flexible and outside of engineering dependency.
